Documentation

  • POST /api/login

    Body

      username

      The username of the user logging in

      password

      The password of the user logging in

    Result on Success

      currentUserID

      The ID of the user

      currentUsername

      The username of the user

  • POST /api/register

    Body

      username

      The username of the user registering

      password

      The password of the user registering

    Result on Success

      currentUserID

      The ID of the user

      currentUsername

      The username of the user

  • /api/get-contact-list

    Body

      currentUserID

      The ID of the currently logged in user

    Result on Success

      Contact[]

      A list of contacts containing

        userID

        The ID of the contact user

        username

        The username of the contact user

        nickname

        The nickname of the contact user given by current user

  • /api/get-contact-info

    Body

      currentUserID

      The ID of the currently logged in user

      contactID

      The ID of the contact user

    Result on Success

      Contact

      A contact containing

        userID

        The ID of the contact user

        username

        The username of the contact user

        nickname

        The nickname of the contact user given by current user

        notes

        The notes added to the contact user by current user

  • /api/create-contact

    Body

      currentUserID

      The ID of the user currently logged in

      contactID or contactUsername

      The ID or the username of the user to add to a contact

      nickname (Optional)

      The nickname to assign to the contact user

      notes (Optional)

      Notes to add to the contact user

    Result on Success

      Nothing

      Nothing is returned to the client
